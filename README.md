<h2>🚀 Overview</h2>
<p>
  HTTP Debugger Lite is a modern Windows desktop application built with Python and PySide6
  that captures, inspects, and visualizes live HTTP and HTTPS traffic in real time.
</p>

<p>
  Version <b>v1.2</b> introduces major improvements including a fully packaged installer,
  automatic dependency handling, enhanced UI performance, and advanced response rendering.
</p>

<hr>

<h2>🔥 What's New in v1.2</h2>

<ul>
  <li>💿 Full Windows installer with branding</li>
  <li>⚙️ Automatic Microsoft VC++ Runtime download & install</li>
  <li>⚡ Cython-compiled core modules (enhanced protection)</li>
  <li>🚀 Faster UI updates (no flicker, incremental rendering)</li>
  <li>🖥️ Improved table layout (auto stretch, responsive columns)</li>
  <li>🌐 HTML preview rendering (real website display)</li>
  <li>🖼️ Image preview support (PNG/JPG auto render)</li>
  <li>🔍 Search inside request & response tabs</li>
  <li>📋 Copy buttons + quick actions</li>
  <li>📤 Export support (JSON / XML)</li>
</ul>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>Live HTTP and HTTPS traffic capture</li>
  <li>Automatic proxy enable / disable inside the app</li>
  <li>MITM-based HTTPS inspection</li>
  <li>Real-time request table with:
    <ul>
      <li>ID</li>
      <li>Method</li>
      <li>Status</li>
      <li>URL</li>
      <li>Type</li>
      <li>Size</li>
      <li>Duration</li>
    </ul>
  </li>
  <li>Request Details:
    <ul>
      <li>Headers</li>
      <li>Body</li>
      <li>Raw view</li>
    </ul>
  </li>
  <li>Response Details:
    <ul>
      <li>Headers</li>
      <li>Content</li>
      <li>Raw</li>
      <li>HTML Preview (live rendering)</li>
    </ul>
  </li>
  <li>Pretty JSON viewer with formatting</li>
  <li>Embedded browser preview (Qt WebEngine)</li>
  <li>Image preview support</li>
  <li>Search & filtering system</li>
  <li>Right-click actions + copy tools</li>
  <li>Dark neon cyber-style UI</li>
</ul>

<h2>⚙️ How It Works</h2>

<ol>
  <li>Starts a local proxy backend</li>
  <li>Automatically enables Windows proxy</li>
  <li>Intercepts HTTPS traffic via MITM</li>
  <li>Displays sessions in real time</li>
  <li>Restores proxy when stopped</li>
</ol>
